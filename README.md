📘 SAP ABAP Learning Portfolio
👋 About This Repository

This repository documents my self-study journey in SAP ABAP programming.
It includes simple ABAP programs, ALV reports, and BAPI examples created during my learning process.

💡 The goal is not production code, but to demonstrate my understanding of ABAP syntax, data handling, and SAP development concepts.


🧠 Topics I’m Studying

ABAP syntax and data types (DATA, TYPES, PARAMETERS, SELECT-OPTIONS)

Internal Tables and Looping

ALV Grid Display (CL_GUI_ALV_GRID)

BAPI calls (e.g. BAPI_CUSTOMER_GETDETAIL)

Modularization: FORM, FUNCTION MODULE, CLASS

Performance optimization (SELECT ... FOR ALL ENTRIES, FIELD-SYMBOLS)
